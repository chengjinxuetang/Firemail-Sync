# Building Mailsync

WARNING: The build system is brittle at the moment, and is not likely to work
outside of very specific environments. We're working on fixing that.

## Dependencies

On Ubuntu, the following libraries must be installed:

* libc-ares-dev
* libicu-dev
* libctemplate-dev
* libtidy-dev
* uuid-dev
* libxml2-dev
* libsasl2-dev
* liblzma-dev
